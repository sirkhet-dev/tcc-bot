# Security Policy

## Supported Versions

Only the latest `main` branch is actively supported with security fixes.

## Reporting a Vulnerability

- Please do not disclose security issues publicly.
- Report privately to `dev@sirkhet.com` with reproduction details.

## Security Baseline

- Telegram access is restricted to `ALLOWED_USER_ID`.
- Prompt size and per-user message rate limits are enforced.
- Project discovery is limited to Git repositories under `WORKSPACE_ROOT`.
